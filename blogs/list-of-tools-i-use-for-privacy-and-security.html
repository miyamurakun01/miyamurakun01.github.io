List of tools I use for privacy and security

System

  • Arch Linux + penguinRice - main OS
  • Android - Mobile OS
  • Tails, Whonix - Privacy OS with system-wide Tor

Browser

  • Librewolf - Firefox but better
  • Tor Browser - The best privacy browser but it is slow af
  • uBlock Origin - Content blocker
  • Multi-Account Containers - For multi account

Mail

  • Proton Mail - Mail provider
  • K-9 Mail - Mail client on Android

VPN

  • Tor
  • Riseup VPN
  • Orbot

Search engine

  • DuckDuckGo
  • SearX
  • Startpage

Password manager (btw i dont use password manager anymore)

  • Bitwarden (cloud sync)
  • KeePass, KeePassXC, KeePassDX (local)

File sharing

  • Tim Visee's Send - Fork of Firefox Send
  • Syncthing

Frontends

  • Newpipe (YouTube)
