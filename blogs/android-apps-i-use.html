Android apps I use

[Last edited: Aug 21 2023 10:47 PM]

General

  • F-Droid - An open-source app store for FOSS apps.
  • Aurora Store - Google Play alternative.
  • Termux - A terminal emulator for Android with Linux environment, i can't live without Linux.
  • ZArchiver - Archive manager

Network

  • Mull - A good browser, basically a hardened version of Firefox Android.
  • RiseupVPN - A good VPN. It's open-source, free and private. I use it when my ISP block some websites.
  • Orbot - Tor system-wide. I seldom use it.
  • Proton Mail - A mail provider.

Security and privacy

  • Shizuku - Root permissions without root.
  • Shelter and Island - Isolated space to clone apps and protect privacy.
  • Aegis - Two factor authentication.

Media

  • mpv - Media player with hardware accelration.
  • Tachiyomi - Manga reader.
  • AL-chan - An unofficial client for Anilist, a place where you can track, share, discover, and experience Anime and Manga.
  • Newpipe - YouTube frontend.

Games

  • osu! and osu!droid - rhythm is just a *click* away.

Customization

  • Nova Launcher - The best launcher on Android.
  • Delta Icon Pack - A pastel icon pack.

Other

  • Simple Mobile Tools - You can install the Pro version on F-Droid
  • Discord
  • GitHub

Why I use FOSS apps?

  • Lightweight
  • Privacy-respected
  • Independent (because most of them are libre software)
