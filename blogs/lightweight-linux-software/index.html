Lightweight software for Linux to save your old computer

Distros

  • Arch Linux: A lightweight and simple Linux distro.
  • Void Linux: Simple Linux distro without systemd.
  • Alpine Linux: A small, simple, secure Linux distro without GNU stuffs.
  • Fedora LXDE Spin: Lightweight Linux distro for normal people.
  • Debian: One of the oldest distros but still in active development.

System

Desktop environment/Window manager

  • i3wm: A fully featured tiling window manager
  • bspwm: A small tiling window manager based on binary space partitioning.
  • Openbox: A floating window manager which is used on many DE.
  • IceWM: Other floating window manager.
  • dwm: Suckless window manager.
  • LXDE: The lightest desktop environment.
  • Xfce: A lightweight desktop environment for *nix systems.
  • Lxqt: LXDE + qt (sucks)

Terminal

  • urxvt: lightweight terminal emulator
  • st: suckless terminal.
  • xfce4 terminal: Fully-featured but still lightweight

Internet

Web browser

  • Ungoogled Chromium and Chromium: lighter than firefox and gecko-based
  • qutebrowser: simple browser with vi-keybindings
  • surf: suckless browser
  • Midori: Lightweight browser which uses WebkitGTK engine
  • Falkon: Lightweight QT browser developed by KDE

Email client

  • mutt and neomutt: Terminal mail client
  • Sylpheed: Simple, lighweight but fully-featured mail client
  • Claws Mail: GTK-based mail client

Chat/IRC client

  • Weechat: Terminal IRC client
  • Pidgin: foss chat/irc client
  • HexChat: An IRC client based on XChat
  • Quassel: Qt-based IRC client

Office

  • AbiWord: Lightweight word processor, an alternative for LibreOffce and Microsoft Office
  • Gnumeric: Lightweight spreadsheets program, an alternative for Excel
  • sc-im: An Excel-like terminal spreadsheet manager.
  • Zathura: Lightweight document viewer with vi-keybindings
  • MuPDF: Lightweight document reader

Multimedia

Video player

  • mpv: lightest media player (it can play audio too)
  • vlc: the most popular media player
  • Parole Media Player: Xfce's media player

Music player

  • mpd: music player daemon
  • ncmpcpp: frondend for mpd
  • cmus: terminal music player
  • LXMusic: LXDE's music player
  • moc: console music player
  • musikcube: fully-featured terminal music player

Image viewer

  • feh: x11 image viewer for console users
